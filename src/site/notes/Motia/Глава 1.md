# üìö –ß–æ–º—É –°–∞–º–µ –ü–æ–¥—ñ–π–Ω–æ-–ö–µ—Ä–æ–≤–∞–Ω—ñ –ú—ñ–∫—Ä–æ—Å–µ—Ä–≤—ñ—Å–∏?

–¶—è –≥–ª–∞–≤–∞ –∑–∞–∫–ª–∞–¥–∞—î –∞—Ä—Ö—ñ—Ç–µ–∫—Ç—É—Ä–Ω—ñ –æ—Å–Ω–æ–≤–∏ –¥–ª—è —Ä–æ–∑—É–º—ñ–Ω–Ω—è **–ü–æ–¥—ñ–π–Ω–æ-–∫–µ—Ä–æ–≤–∞–Ω–∏—Ö –º—ñ–∫—Ä–æ—Å–µ—Ä–≤—ñ—Å—ñ–≤ (Event-Driven Microservices, EDM)**, —Å—Ç–≤–µ—Ä–¥–∂—É—é—á–∏, —â–æ —Ü–µ –Ω–µ –ø—Ä–æ—Å—Ç–æ –∞—Å–∏–Ω—Ö—Ä–æ–Ω–Ω–∞ –∑–∞–º—ñ–Ω–∞ —Å–∏–Ω—Ö—Ä–æ–Ω–Ω–∏–º —Å–∏—Å—Ç–µ–º–∞–º, –∞ **—Ñ—É–Ω–¥–∞–º–µ–Ω—Ç–∞–ª—å–Ω–æ —ñ–Ω—à–∏–π —Å–ø–æ—Å—ñ–± –æ—Ä–≥–∞–Ω—ñ–∑–∞—Ü—ñ—ó –¥—ñ—è–ª—å–Ω–æ—Å—Ç—ñ –ø—ñ–¥–ø—Ä–∏—î–º—Å—Ç–≤–∞** [1, 2].

## 1. –§—É–Ω–¥–∞–º–µ–Ω—Ç–∞–ª—å–Ω—ñ –ü—Ä–∏–Ω—Ü–∏–ø–∏ EDM

EDM-–∞—Ä—Ö—ñ—Ç–µ–∫—Ç—É—Ä–∞ –Ω–∞–±—É–ª–∞ –≤–ø–ª–∏–≤—É –∑–∞–≤–¥—è–∫–∏ –∫–æ–Ω–≤–µ—Ä–≥–µ–Ω—Ü—ñ—ó **–≤–µ–ª–∏–∫–∏—Ö –¥–∞–Ω–∏—Ö** —Ç–∞ –ø–æ—Ç—Ä–µ–±—ñ –æ–±—Ä–æ–±–ª—è—Ç–∏ —ñ–Ω—Ñ–æ—Ä–º–∞—Ü—ñ—é –º–∞–π–∂–µ –≤ —Ä–µ–∞–ª—å–Ω–æ–º—É —á–∞—Å—ñ [3].

### 1.1. –í—ñ–¥–º—ñ–Ω–Ω—ñ—Å—Ç—å –≤—ñ–¥ –û–±–º—ñ–Ω—É –ü–æ–≤—ñ–¥–æ–º–ª–µ–Ω–Ω—è–º–∏
–ì–æ–ª–æ–≤–Ω–∞ –≤—ñ–¥–º—ñ–Ω–Ω—ñ—Å—Ç—å EDM –≤—ñ–¥ —Ç—Ä–∞–¥–∏—Ü—ñ–π–Ω–∏—Ö —Å–∏—Å—Ç–µ–º –ø–µ—Ä–µ–¥–∞—á—ñ –ø–æ–≤—ñ–¥–æ–º–ª–µ–Ω—å –ø–æ–ª—è–≥–∞—î —É —Å—Ç–∞–≤–ª–µ–Ω–Ω—ñ –¥–æ –ø–æ–¥—ñ–π:
*   –£ EDM-—Å–∏—Å—Ç–µ–º–∞—Ö –ø–æ–¥—ñ—ó **–Ω–µ –∑–Ω–∏—â—É—é—Ç—å—Å—è** –ø—ñ—Å–ª—è —Å–ø–æ–∂–∏–≤–∞–Ω–Ω—è, –∞ –∑–∞–ª–∏—à–∞—é—Ç—å—Å—è –ª–µ–≥–∫–æ –¥–æ—Å—Ç—É–ø–Ω–∏–º–∏ –¥–ª—è —ñ–Ω—à–∏—Ö —Å–ø–æ–∂–∏–≤–∞—á—ñ–≤ [4].
*   –ü–æ–¥—ñ—ó –º–æ–∂—É—Ç—å –∑–±–µ—Ä—ñ–≥–∞—Ç–∏—Å—è **–Ω–µ—Å–∫—ñ–Ω—á–µ–Ω–Ω–æ** (–∞–±–æ —É –Ω–∞–¥–∑–≤–∏—á–∞–π–Ω–æ –≤–µ–ª–∏–∫–∏—Ö –º–∞—Å—à—Ç–∞–±–∞—Ö) [4].

–ö–æ–º–±—ñ–Ω–∞—Ü—ñ—è [[–ü–æ—Ç–æ–∫–∏ –ü–æ–¥—ñ–π\|–ü–æ—Ç–æ–∫–∏ –ü–æ–¥—ñ–π]] —Ç–∞ –º–∞–ª–∏—Ö, —Å–ø–µ—Ü—ñ–∞–ª—ñ–∑–æ–≤–∞–Ω–∏—Ö –º—ñ–∫—Ä–æ—Å–µ—Ä–≤—ñ—Å—ñ–≤ —É—Ç–≤–æ—Ä—é—î –≤–∑–∞—î–º–æ–ø–æ–≤'—è–∑–∞–Ω–∏–π **–≥—Ä–∞—Ñ –∞–∫—Ç–∏–≤–Ω–æ—Å—Ç—ñ** –≤—Å—ñ—î—ó –¥—ñ–ª–æ–≤–æ—ó –æ—Ä–≥–∞–Ω—ñ–∑–∞—Ü—ñ—ó [5].

### 1.2. –°–ª–∞–±–∫–∞ –ó–≤'—è–∑–∞–Ω—ñ—Å—Ç—å
–î–ª—è –µ—Ñ–µ–∫—Ç–∏–≤–Ω–æ—ó —Ä–æ–±–æ—Ç–∏ EDM-—Å–∏—Å—Ç–µ–º –∂–∏—Ç—Ç—î–≤–æ –≤–∞–∂–ª–∏–≤–∏–π –ø—Ä–∏–Ω—Ü–∏–ø **—Å–ª–∞–±–∫–æ—ó –∑–≤'—è–∑–∞–Ω–æ—Å—Ç—ñ** (*—Å–ª–∞–±–∞—è —Å—Ü–µ–ø–ª–µ–Ω–Ω–æ—Å—Ç—å*) [6, 7].
*   EDM-–º—ñ–∫—Ä–æ—Å–µ—Ä–≤—ñ—Å–∏ –∑—á–µ–ø–ª–µ–Ω—ñ –∑ **–¥–∞–Ω–∏–º–∏ –ø—Ä–µ–¥–º–µ—Ç–Ω–æ—ó –æ–±–ª–∞—Å—Ç—ñ**, –∞ –Ω–µ –∑ API –∫–æ–Ω–∫—Ä–µ—Ç–Ω–æ—ó —Ä–µ–∞–ª—ñ–∑–∞—Ü—ñ—ó [7].
*   –¶–µ –≥–∞—Ä–∞–Ω—Ç—É—î, —â–æ –∑–º—ñ–Ω–∞ –≤–∏–º–æ–≥ –≤ –æ–¥–Ω–æ–º—É –∫–æ–Ω—Ç–µ–∫—Å—Ç—ñ **–Ω–µ –≤–∏–º–∞–≥–∞—Ç–∏–º–µ –≤–Ω–µ—Å–µ–Ω–Ω—è –∑–∞–ª–µ–∂–Ω–∏—Ö –∑–º—ñ–Ω** —É —Å—É—Å—ñ–¥–Ω—ñ –∫–æ–Ω—Ç–µ–∫—Å—Ç–∏ [6].

## 2. –ü—Ä–µ–¥–º–µ—Ç–Ω–æ-–û—Ä—ñ—î–Ω—Ç–æ–≤–∞–Ω–µ –ü—Ä–æ—î–∫—Ç—É–≤–∞–Ω–Ω—è (DDD)

–ü–æ–±—É–¥–æ–≤–∞ –º—ñ–∫—Ä–æ—Å–µ—Ä–≤—ñ—Å—ñ–≤ –≤–∏–º–∞–≥–∞—î –∑–∞—Å—Ç–æ—Å—É–≤–∞–Ω–Ω—è –∫–æ–Ω—Ü–µ–ø—Ü—ñ–π [[–ü—Ä–µ–¥–º–µ—Ç–Ω–æ-–æ—Ä—ñ—î–Ω—Ç–æ–≤–∞–Ω–µ –ü—Ä–æ—î–∫—Ç—É–≤–∞–Ω–Ω—è (DDD)\|–ü—Ä–µ–¥–º–µ—Ç–Ω–æ-–æ—Ä—ñ—î–Ω—Ç–æ–≤–∞–Ω–µ –ü—Ä–æ—î–∫—Ç—É–≤–∞–Ω–Ω—è (DDD)]] –¥–ª—è —É–∑–≥–æ–¥–∂–µ–Ω–Ω—è —Ç–µ—Ö–Ω—ñ—á–Ω–æ—ó —Ä–µ–∞–ª—ñ–∑–∞—Ü—ñ—ó –∑ –ø–æ—Ç—Ä–µ–±–∞–º–∏ –±—ñ–∑–Ω–µ—Å—É [8].

### 2.1. –ö–ª—é—á–æ–≤—ñ –ö–æ–Ω—Ü–µ–ø—Ü—ñ—ó
*   **–î–æ–º–µ–Ω (Domain):** –¶–µ –ø—Ä–µ–¥–º–µ—Ç–Ω–∞ –æ–±–ª–∞—Å—Ç—å, —É —è–∫—ñ–π –ø—Ä–∞—Ü—é—î –±—ñ–∑–Ω–µ—Å, –≤–∫–ª—é—á–Ω–æ –∑ –π–æ–≥–æ –ø—Ä–∞–≤–∏–ª–∞–º–∏, –ø—Ä–æ—Ü–µ—Å–∞–º–∏ —Ç–∞ —Ç–µ—Ä–º—ñ–Ω–æ–ª–æ–≥—ñ—î—é [8].
*   **–ú–æ–¥–µ–ª—å –ü—Ä–µ–¥–º–µ—Ç–Ω–æ—ó –û–±–ª–∞—Å—Ç—ñ (Domain Model):** –¶–µ –∫–æ–Ω—Å—Ç—Ä—É–∫—Ü—ñ—è, —è–∫—É –ø—ñ–¥–ø—Ä–∏—î–º—Å—Ç–≤–æ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—î –¥–ª—è –≤–∏—Ä—ñ—à–µ–Ω–Ω—è —Å–≤–æ—ó—Ö –∑–∞–≤–¥–∞–Ω—å (—á–∞—Å—Ç–∏–Ω–∞ –ø—Ä–æ—Å—Ç–æ—Ä—É —Ä—ñ—à–µ–Ω—å) [9].
*   **[[–û–±–º–µ–∂–µ–Ω–∏–π –ö–æ–Ω—Ç–µ–∫—Å—Ç\|–û–±–º–µ–∂–µ–Ω–∏–π –ö–æ–Ω—Ç–µ–∫—Å—Ç]] (Bounded Context):** –õ–æ–≥—ñ—á–Ω—ñ –º–µ–∂—ñ, —è–∫—ñ –æ—Ö–æ–ø–ª—é—é—Ç—å –≤—Ö–æ–¥–∏, –≤–∏—Ö–æ–¥–∏, –ø–æ–¥—ñ—ó, –≤–∏–º–æ–≥–∏ —Ç–∞ –º–æ–¥–µ–ª—ñ –¥–∞–Ω–∏—Ö –ø—ñ–¥–¥–æ–º–µ–Ω—É [6]. –í–æ–Ω–∏ —î –æ—Å–Ω–æ–≤–æ—é –¥–ª—è —Å—Ç–≤–æ—Ä–µ–Ω–Ω—è –Ω–µ–∑–∞–ª–µ–∂–Ω–∏—Ö –º—ñ–∫—Ä–æ—Å–µ—Ä–≤—ñ—Å—ñ–≤ [10, 11].

### 2.2. –ü—Ä—ñ–æ—Ä–∏—Ç–µ—Ç –ë—ñ–∑–Ω–µ—Å-–í–∏–º–æ–≥
–ö—Ä–∏—Ç–∏—á–Ω–æ –≤–∞–∂–ª–∏–≤–æ –ø—Ä–∏–≤'—è–∑—É–≤–∞—Ç–∏ –æ–±–º–µ–∂–µ–Ω—ñ –∫–æ–Ω—Ç–µ–∫—Å—Ç–∏ –¥–æ **–±—ñ–∑–Ω–µ—Å-–≤–∏–º–æ–≥**, –∞ –Ω–µ –¥–æ —Ç–µ—Ö–Ω–æ–ª–æ–≥—ñ—á–Ω–∏—Ö [6]. –¶–µ –Ω–∞–¥–∞—î –∫–æ–º–∞–Ω–¥—ñ **–∞–≤—Ç–æ–Ω–æ–º—ñ—é** –≤ –¥–∏–∑–∞–π–Ω—ñ —Ç–∞ —Ä–µ–∞–ª—ñ–∑–∞—Ü—ñ—ó —Ä—ñ—à–µ–Ω–Ω—è [6]. –ü—Ä–∏–≤'—è–∑–∫–∞ –¥–æ —Ç–µ—Ö–Ω—ñ—á–Ω–∏—Ö –≤–∏–º–æ–≥ —Å—Ç–≤–æ—Ä—é—î –ø—Ä–æ–±–ª–µ–º–∏, –æ—Å–∫—ñ–ª—å–∫–∏ –∑–º—ñ–Ω–∞ API —Å—Ç–∞—î **–¥–æ—Ä–æ–≥–µ–Ω—å–∫–æ—é** (*–¥–æ—Ä–æ–≥–æ—Å—Ç–æ—è—â–µ–π*) —Ç–∞ –º–æ–∂–µ –≤–ø–ª–∏–Ω—É—Ç–∏ –Ω–∞ –º–æ–∂–ª–∏–≤–æ—Å—Ç—ñ —Ä–æ–±–æ—Ç–∏ –≤—Å—å–æ–≥–æ –ø—ñ–¥–ø—Ä–∏—î–º—Å—Ç–≤–∞ [12].

## 3. –ü–æ—Ç–æ–∫–∏ –ü–æ–¥—ñ–π —è–∫ –Ñ–¥–∏–Ω–µ –î–∂–µ—Ä–µ–ª–æ –Ü—Å—Ç–∏–Ω–∏ (SSOT)

–¢—Ä–∞–¥–∏—Ü—ñ–π–Ω—ñ [[–°—Ç—Ä—É–∫—Ç—É—Ä–∏ –û–±–º—ñ–Ω—É –¢–µ—Ö–Ω—ñ—á–Ω–æ—é –Ü–Ω—Ñ–æ—Ä–º–∞—Ü—ñ—î—é\|–°—Ç—Ä—É–∫—Ç—É—Ä–∏ –û–±–º—ñ–Ω—É –¢–µ—Ö–Ω—ñ—á–Ω–æ—é –Ü–Ω—Ñ–æ—Ä–º–∞—Ü—ñ—î—é]] (–Ω–∞–ø—Ä–∏–∫–ª–∞–¥, —Å–ø—ñ–ª—å–Ω—ñ –±–∞–∑–∏ –¥–∞–Ω–∏—Ö –∞–±–æ –ø—Ä—è–º—ñ –∑–∞–ø–∏—Ç–∏) –ø—Ä–∏–∑–≤–æ–¥—è—Ç—å –¥–æ **—Å–∏–ª—å–Ω–æ—ó —Å—Ü–µ–ø–∫–∏** ¬´—Ç–æ—á–∫–∞-—Ç–æ—á–∫–∞¬ª (*—Ç–æ—á–∫–∞-—Ç–æ—á–∫–∞*) [13, 14].

### 3.1. –Ü–Ω–≤–µ—Ä—Å—ñ—è –ú–æ–¥–µ–ª—ñ
EDM –∑–∞–º—ñ—â—É—î –Ω–µ—Ä–µ–≥–ª–∞–º–µ–Ω—Ç–æ–≤–∞–Ω—É —Å—Ç—Ä—É–∫—Ç—É—Ä—É **—Ñ–æ—Ä–º–∞–ª—ñ–∑–æ–≤–∞–Ω–æ—é —Å—Ç—Ä—É–∫—Ç—É—Ä–æ—é –æ–±–º—ñ–Ω—É –¥–∞–Ω–∏–º–∏** [15].
*   **–£–Ω—ñ–≤–µ—Ä—Å–∞–ª—å–Ω–∏–π –î–æ—Å—Ç—É–ø:** –°—Ç—Ä–∏–∂–Ω–µ–≤—ñ –±—ñ–∑–Ω–µ—Å-–¥–∞–Ω—ñ –º–∞—é—Ç—å –±—É—Ç–∏ **–ª–µ–≥–∫–æ –¥–æ—Å—Ç—É–ø–Ω–∏–º–∏** –¥–ª—è –±—É–¥—å-—è–∫–æ–≥–æ —Å–µ—Ä–≤—ñ—Å—É, —è–∫–∏–π —ó—Ö –ø–æ—Ç—Ä–µ–±—É—î [15].
*   **–†–æ–∑–º–µ–∂—É–≤–∞–Ω–Ω—è:** –í–∏—Ä–æ–±–Ω–∏—Ü—Ç–≤–æ —Ç–∞ –≤–æ–ª–æ–¥—ñ–Ω–Ω—è –¥–∞–Ω–∏–º–∏ **–ø–æ–≤–Ω—ñ—Å—Ç—é —Ä–æ–∑–º–µ–∂–æ–≤–∞–Ω—ñ** –≤—ñ–¥ —ó—Ö —Å–ø–æ–∂–∏–≤–∞–Ω–Ω—è [16]. –¶–µ —É—Å—É–≤–∞—î –Ω–µ–æ–±—Ö—ñ–¥–Ω—ñ—Å—Ç—å –¥–ª—è –≤–∏—Ä–æ–±–Ω–∏–∫–∞ –Ω–∞–¥–∞–≤–∞—Ç–∏ API –∞–±–æ —Ç—Ä–∞–Ω—Å–≥—Ä–∞–Ω–∏—á–Ω—ñ —Å–µ—Ä–≤—ñ—Å–∏ –¥–ª—è –æ–±–º—ñ–Ω—É –¥–∞–Ω–∏–º–∏ [17].

### 3.2. –Ñ–¥–∏–Ω–µ –î–∂–µ—Ä–µ–ª–æ –Ü—Å—Ç–∏–Ω–∏
–ö–æ–∂–Ω–∞ –ø–æ–¥—ñ—è —É –ø–æ—Ç–æ—Ü—ñ ‚Äî —Ü–µ **–∫–æ–Ω—Å—Ç–∞—Ç–∞—Ü—ñ—è —Ñ–∞–∫—Ç—É** (*–∫–æ–Ω—Å—Ç–∞—Ç–∞—Ü–∏—è —Ñ–∞–∫—Ç–∞*) —Ç–æ–≥–æ, —â–æ —Å—Ç–∞–ª–æ—Å—è [18]. –†–∞–∑–æ–º —Ü—ñ –∫–æ–Ω—Å—Ç–∞—Ç–∞—Ü—ñ—ó —Ñ–æ—Ä–º—É—é—Ç—å [[–Ñ–¥–∏–Ω–µ –î–∂–µ—Ä–µ–ª–æ –Ü—Å—Ç–∏–Ω–∏ (SSOT)\|–Ñ–¥–∏–Ω–µ –î–∂–µ—Ä–µ–ª–æ –Ü—Å—Ç–∏–Ω–∏ (SSOT)]], —è–∫–µ —î –ø–æ–≤–Ω–æ—é —ñ—Å—Ç–æ—Ä—ñ—î—é [18, 19].

### 3.3. –†–∏–∑–∏–∫–∏ –¢—Ä–∞–¥–∏—Ü—ñ–π–Ω–æ–≥–æ –ü—ñ–¥—Ö–æ–¥—É
–í—ñ–¥—Å—É—Ç–Ω—ñ—Å—Ç—å –Ω–∞–ª–µ–∂–Ω–æ—ó —Å—Ç—Ä—É–∫—Ç—É—Ä–∏ –æ–±–º—ñ–Ω—É –¥–∞–Ω–∏–º–∏ –ø—Ä–∏–∑–≤–æ–¥–∏—Ç—å –¥–æ:
*   –ö–æ–ø—ñ—é–≤–∞–Ω–Ω—è –¥–∞–Ω–∏—Ö, —è–∫—ñ –∑–∞–≤–∂–¥–∏ –±—É–¥—É—Ç—å **–¥–µ—â–æ –∑–∞—Å—Ç–∞—Ä—ñ–ª–∏–º–∏** (*–Ω–µ—Å–∫–æ–ª—å–∫–æ —É—Å—Ç–∞—Ä–µ–≤—à–∏–º–∏*) [20, 21].
*   –ù–µ–æ–±—Ö—ñ–¥–Ω–æ—Å—Ç—ñ –ø–æ–∫–ª–∞–¥–∞—Ç–∏—Å—è –Ω–∞ **¬´–ø–ª–µ–º—ñ–Ω–Ω—ñ –∑–Ω–∞–Ω–Ω—è¬ª** (*–ø–ª–µ–º–µ–Ω–Ω–æ–µ –∑–Ω–∞–Ω–∏–µ*) –¥–ª—è —ñ–Ω—Ç–µ—Ä–ø—Ä–µ—Ç–∞—Ü—ñ—ó –¥–∞–Ω–∏—Ö, —â–æ –Ω–µ –º–∞—Å—à—Ç–∞–±—É—î—Ç—å—Å—è [21, 22].

## 4. –ü–æ—Ä—ñ–≤–Ω—è–Ω–Ω—è –∑ –°–∏–Ω—Ö—Ä–æ–Ω–Ω–∏–º–∏ –ú—ñ–∫—Ä–æ—Å–µ—Ä–≤—ñ—Å–∞–º–∏

–•–æ—á–∞ —Å–∏–Ω—Ö—Ä–æ–Ω–Ω—ñ –º—ñ–∫—Ä–æ—Å–µ—Ä–≤—ñ—Å–∏ (*–∑–∞–ø—Ä–æ—Å-–æ—Ç–≤–µ—Ç*) –º–∞—é—Ç—å —Å–≤–æ—î –º—ñ—Å—Ü–µ [23], –≤–æ–Ω–∏ –Ω–µ—Å—É—Ç—å –∑–Ω–∞—á–Ω—ñ –Ω–µ–¥–æ–ª—ñ–∫–∏, —è–∫—ñ –≤–∏—Ä—ñ—à—É—î EDM:
