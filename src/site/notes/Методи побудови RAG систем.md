**–©–æ —Ä–æ–±–∏—Ç—å –≤–µ–∫—Ç–æ—Ä–Ω–∞ –±–∞–∑–∞ –¥–∞–Ω–∏—Ö?**

- –ó–±–µ—Ä—ñ–≥–∞–Ω–Ω—è —Ç–µ–∫—Å—Ç–æ–≤–∏—Ö —Ñ—Ä–∞–≥–º–µ–Ω—Ç—ñ–≤ —É –≤–∏–≥–ª—è–¥—ñ –±–∞–≥–∞—Ç–æ–≤–∏–º—ñ—Ä–Ω–∏—Ö –≤–µ–∫—Ç–æ—Ä—ñ–≤.
- –ü–æ—à—É–∫ –Ω–∞–π–±—ñ–ª—å—à –ø–æ–¥—ñ–±–Ω–∏—Ö –≤–µ–∫—Ç–æ—Ä—ñ–≤ –∑–∞ –∑–∞–ø–∏—Ç–æ–º.
- –í–∏–∫–æ—Ä–∏—Å—Ç–∞–Ω–Ω—è —ñ–Ω–¥–µ–∫—Å—ñ–≤ –¥–ª—è —à–≤–∏–¥–∫–æ–≥–æ –ø–æ—à—É–∫—É (ANN ‚Äî Approximate Nearest Neighbors).

**–û—Å–Ω–æ–≤–Ω—ñ –∫—Ä–∏—Ç–µ—Ä—ñ—ó –ø—Ä–∏ –≤–∏–±–æ—Ä—ñ:**

1. **–†–æ–∑–º—ñ—Ä –¥–∞–Ω–∏—Ö —Ç–∞ –º–∞—Å—à—Ç–∞–±–æ–≤–∞–Ω—ñ—Å—Ç—å**
   - –ù–µ–≤–µ–ª–∏–∫–∏–π –æ–±—Å—è–≥ –¥–∞–Ω–∏—Ö ‚Üí FAISS, Chroma
   - –Ø–∫—â–æ –ø–æ—Ç—Ä—ñ–±–µ–Ω –º–∞—Å—à—Ç–∞–±–æ–≤–∞–Ω–∏–π –ø–æ—à—É–∫ ‚Üí Weaviate, Milvus, Qdrant

2. **–®–≤–∏–¥–∫—ñ—Å—Ç—å –ø–æ—à—É–∫—É**
   - –Ø–∫—â–æ –≤–∞–∂–ª–∏–≤–∞ —à–≤–∏–¥–∫–∞ –æ–±—Ä–æ–±–∫–∞ ‚Üí FAISS, ScaNN (Google)
   - –Ø–∫—â–æ –≤–∞–∂–ª–∏–≤–∏–π –æ–Ω–ª–∞–π–Ω-–¥–æ—Å—Ç—É–ø —Ç–∞ REST API ‚Üí Weaviate, Pinecone

3. **–ü—ñ–¥—Ç—Ä–∏–º–∫–∞ –≥—ñ–±—Ä–∏–¥–Ω–æ–≥–æ –ø–æ—à—É–∫—É (–≤–µ–∫—Ç–æ—Ä–Ω–∏–π + –∫–ª–∞—Å–∏—á–Ω–∏–π full-text)**
   - Weaviate, Qdrant, Vespa –¥–æ–∑–≤–æ–ª—è—é—Ç—å –∫–æ–º–±—ñ–Ω—É–≤–∞—Ç–∏ –≤–µ–∫—Ç–æ—Ä–Ω–∏–π —Ç–∞ —Ç–µ–∫—Å—Ç–æ–≤–∏–π –ø–æ—à—É–∫.

4. **–ü—Ä–æ—Å—Ç–æ—Ç–∞ —ñ–Ω—Ç–µ–≥—Ä–∞—Ü—ñ—ó**
   - FAISS, Chroma ‚Äî –ª–µ–≥–∫–æ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ –≤ Python.
   - Pinecone, Weaviate, Qdrant ‚Äî –∑—Ä—É—á–Ω—ñ –¥–ª—è —Ö–º–∞—Ä–Ω–∏—Ö —Ä—ñ—à–µ–Ω—å.

**–ü—Ä–∏–∫–ª–∞–¥ –ø–æ—à—É–∫—É —Å—Ö–æ–∂–∏—Ö –≤–µ–∫—Ç–æ—Ä—ñ–≤:**

```python
import faiss
import numpy as np
from sentence_transformers import SentenceTransformer

# –ó–∞–≤–∞–Ω—Ç–∞–∂—É—î–º–æ –º–æ–¥–µ–ª—å –≤—Å—Ç—Ä–∞–∏–≤–∞–Ω–Ω—è
model = SentenceTransformer("all-MiniLM-L6-v2")

# –Ü—Å—Ö–æ–¥–Ω—ñ —Ç–µ–∫—Å—Ç–æ–≤—ñ —Ñ—Ä–∞–≥–º–µ–Ω—Ç–∏
texts = [
    "–Ø–∫ –ø—Ä–∞—Ü—é—î RAG?",
    "–í–µ–∫—Ç–æ—Ä–Ω—ñ –±–∞–∑–∏ –¥–∞–Ω–∏—Ö –∑–±–µ—Ä—ñ–≥–∞—é—Ç—å embeddings.",
    "–ú–∞—à–∏–Ω–Ω–µ –Ω–∞–≤—á–∞–Ω–Ω—è –ø–æ–∫—Ä–∞—â—É—î –ø–æ—à—É–∫ —ñ–Ω—Ñ–æ—Ä–º–∞—Ü—ñ—ó.",
    "–ì–ª–∏–±–æ–∫—ñ –Ω–µ–π—Ä–æ–º–µ—Ä–µ–∂—ñ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—é—Ç—å—Å—è –≤ NLP."
]

# –ü–µ—Ä–µ—Ç–≤–æ—Ä—é—î–º–æ —Ç–µ–∫—Å—Ç —É –≤–µ–∫—Ç–æ—Ä–∏
embeddings = model.encode(texts)
dimension = embeddings.shape[1]  # –í–∏–∑–Ω–∞—á–∞—î–º–æ —Ä–æ–∑–º—ñ—Ä–Ω—ñ—Å—Ç—å –≤–µ–∫—Ç–æ—Ä—ñ–≤

# –°—Ç–≤–æ—Ä—é—î–º–æ —ñ–Ω–¥–µ–∫—Å FAISS
index = faiss.IndexFlatL2(dimension)
index.add(np.array(embeddings))  # –î–æ–¥–∞—î–º–æ –≤–µ–∫—Ç–æ—Ä–∏ –≤ –±–∞–∑—É

# –ó–∞–ø–∏—Ç (—Ç–µ–∂ –ø–µ—Ä–µ—Ç–≤–æ—Ä—é—î–º–æ –≤ –≤–µ–∫—Ç–æ—Ä)
query = "–Ø–∫ –ø—Ä–∞—Ü—é—é—Ç—å –≤–µ–∫—Ç–æ—Ä–Ω—ñ –±–∞–∑–∏?"
query_embedding = model.encode([query])

# –®—É–∫–∞—î–º–æ 2 –Ω–∞–π–±–ª–∏–∂—á–∏—Ö –≤–µ–∫—Ç–æ—Ä–∞
D, I = index.search(np.array(query_embedding), k=2)

# –í–∏–≤–æ–¥–∏–º–æ —Ä–µ–∑—É–ª—å—Ç–∞—Ç–∏
print("–ó–Ω–∞–π–¥–µ–Ω—ñ —Ñ—Ä–∞–≥–º–µ–Ω—Ç–∏:")
for idx in I[0]:
    print(f"- {texts[idx]}")
```

**–†–µ–∑—É–ª—å—Ç–∞—Ç –≤–∏–∫–æ–Ω–∞–Ω–Ω—è:**

- –ó–Ω–∞–π–¥–µ–Ω—ñ —Ñ—Ä–∞–≥–º–µ–Ω—Ç–∏:
  - –Ø–∫ –ø—Ä–∞—Ü—é—î RAG?
  - –ú–∞—à–∏–Ω–Ω–µ –Ω–∞–≤—á–∞–Ω–Ω—è –ø–æ–∫—Ä–∞—â—É—î –ø–æ—à—É–∫ —ñ–Ω—Ñ–æ—Ä–º–∞—Ü—ñ—ó.

#### –û—Ä–≥–∞–Ω—ñ–∑–∞—Ü—ñ—è –ø–æ—à—É–∫—É —Ç–∞ –ø–µ—Ä–µ—Ä–∞–Ω–∂—É–≤–∞–Ω–Ω—è

–ú–µ—Ç–æ–¥–∏ –ø–æ—à—É–∫—É, —Ç–∞–∫—ñ —è–∫ –≥–µ–Ω–µ—Ä–∞—Ü—ñ—è –∑–∞–ø–∏—Ç—ñ–≤, –∑–Ω–∞—á–Ω–æ –ø–æ–∫—Ä–∞—â—É—é—Ç—å —Ä–µ–ª–µ–≤–∞–Ω—Ç–Ω—ñ—Å—Ç—å —Ä–µ–∑—É–ª—å—Ç–∞—Ç—ñ–≤. –û–¥–Ω–∞–∫ —Å–ª—ñ–¥ –≤—Ä–∞—Ö–æ–≤—É–≤–∞—Ç–∏, —â–æ —Ü—ñ –º–µ—Ç–æ–¥–∏ –º–æ–∂—É—Ç—å –±—É—Ç–∏ —Ä–µ—Å—É—Ä—Å–æ–º—ñ—Å—Ç–∫–∏–º–∏, —â–æ –≤–∞–∂–ª–∏–≤–æ –≤—Ä–∞—Ö–æ–≤—É–≤–∞—Ç–∏ –≤ —Ä–µ–∞–ª—å–Ω–∏—Ö –∑–∞—Å—Ç–æ—Å—É–≤–∞–Ω–Ω—è—Ö.

1. **–ü–æ—à—É–∫ (Retrieval)**
   –ù–∞ —Ü—å–æ–º—É –µ—Ç–∞–ø—ñ —Å–∏—Å—Ç–µ–º–∞ –≤–∏–∫–æ–Ω—É—î –≤–µ–∫—Ç–æ—Ä–Ω–∏–π –ø–æ—à—É–∫ –∞–±–æ –≥—ñ–±—Ä–∏–¥–Ω–∏–π –ø–æ—à—É–∫ (–∫–æ–º–±—ñ–Ω–∞—Ü—ñ—è –≤–µ–∫—Ç–æ—Ä–Ω–æ–≥–æ —Ç–∞ –∫–ª–∞—Å–∏—á–Ω–æ–≥–æ —Ç–µ–∫—Å—Ç–æ–≤–æ–≥–æ –ø–æ—à—É–∫—É).

**–ú–µ—Ç–æ–¥–∏ –ø–æ—à—É–∫—É:**

- **–í–µ–∫—Ç–æ—Ä–Ω–∏–π –ø–æ—à—É–∫ (ANN ‚Äì Approximate Nearest Neighbors):** –ø–æ—Ä—ñ–≤–Ω—è–Ω–Ω—è –µ–º–±–µ–¥–¥–∏–Ω–≥—É –∑–∞–ø–∏—Ç—É —Ç–∞ –±–∞–∑–∏ (FAISS, Weaviate, Pinecone).
- **–ö–ª–∞—Å–∏—á–Ω–∏–π –ø–æ—à—É–∫ (BM25):** –ø–æ—à—É–∫ –∑–∞ –∫–ª—é—á–æ–≤–∏–º–∏ —Å–ª–æ–≤–∞–º–∏ (Elasticsearch, Weaviate, Qdrant).
- **–ì—ñ–±—Ä–∏–¥–Ω–∏–π –ø–æ—à—É–∫ (BM25 + ANN):** –ø–æ—î–¥–Ω–∞–Ω–Ω—è –¥–≤–æ—Ö –º–µ—Ç–æ–¥—ñ–≤ –¥–ª—è –ø—ñ–¥–≤–∏—â–µ–Ω–Ω—è —Ç–æ—á–Ω–æ—Å—Ç—ñ.

2. **–ü–µ—Ä–µ—Ä–∞–Ω–∂—É–≤–∞–Ω–Ω—è (Re-ranking)**
   –ü—ñ—Å–ª—è –ø–æ—à—É–∫—É –æ—Ç—Ä–∏–º–∞–Ω—ñ –¥–æ–∫—É–º–µ–Ω—Ç–∏ —Ä–∞–Ω–∂—É—é—Ç—å—Å—è –∑–∞ —Ä–µ–ª–µ–≤–∞–Ω—Ç–Ω—ñ—Å—Ç—é. –ü—Ä–æ—Å—Ç—ñ –º–µ—Ç–æ–¥–∏ (BM25) –º–æ–∂—É—Ç—å –ø–æ–≤–µ—Ä—Ç–∞—Ç–∏ –Ω–µ—Ä–µ–ª–µ–≤–∞–Ω—Ç–Ω—ñ —Ä–µ–∑—É–ª—å—Ç–∞—Ç–∏, —Ç–æ–º—É –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—î—Ç—å—Å—è –¥–æ—Ä–∞–Ω–∂—É–≤–∞–Ω–Ω—è (re-ranking) –∑–∞ –¥–æ–ø–æ–º–æ–≥–æ—é –Ω–µ–π—Ä–æ–Ω–Ω–∏—Ö –º–µ—Ä–µ–∂.

**–ú–æ–¥–µ–ª—ñ —Ä–∞–Ω–∂—É–≤–∞–Ω–Ω—è:**

- **Cross-Encoder (SBERT)** ‚Äì –¥–æ–Ω–∞–≤—á–µ–Ω–∞ –º–æ–¥–µ–ª—å –¥–ª—è –≤–∏–∑–Ω–∞—á–µ–Ω–Ω—è –Ω–∞–π–±—ñ–ª—å—à —Ä–µ–ª–µ–≤–∞–Ω—Ç–Ω–∏—Ö —Ñ—Ä–∞–≥–º–µ–Ω—Ç—ñ–≤.
- **Cohere Rerank** ‚Äì —Ö–º–∞—Ä–Ω–∞ –º–æ–¥–µ–ª—å –¥–ª—è –ø–æ–∫—Ä–∞—â–µ–Ω–Ω—è –ø–æ—à—É–∫—É.
- **OpenAI reranking (GPT)** ‚Äì –¥–æ—Ä–∞–Ω–∂—É–≤–∞–Ω–Ω—è –∑–∞ –¥–æ–ø–æ–º–æ–≥–æ—é GPT.

**–ü—Ä–∏–∫–ª–∞–¥ –ø–æ—à—É–∫—É —Ç–∞ –ø–µ—Ä–µ—Ä–∞–Ω–∂—É–≤–∞–Ω–Ω—è:**

```python
import faiss
import numpy as np
from sentence_transformers import SentenceTransformer, CrossEncoder

# –ó–∞–≤–∞–Ω—Ç–∞–∂—É—î–º–æ –º–æ–¥–µ–ª—ñ
retrieval_model = SentenceTransformer("all-MiniLM-L6-v2")  # –î–ª—è –ø–æ—à—É–∫—É
reranker = CrossEncoder("cross-encoder/ms-marco-MiniLM-L-6-v2")  # –î–ª—è –ø–µ—Ä–µ—Ä–∞–Ω–∂—É–≤–∞–Ω–Ω—è

# –Ü—Å—Ö–æ–¥–Ω—ñ —Ç–µ–∫—Å—Ç–æ–≤—ñ —Ñ—Ä–∞–≥–º–µ–Ω—Ç–∏
documents = [
    "LLM –º–æ–∂—É—Ç—å –≥–µ–Ω–µ—Ä–∏—Ä—É–≤–∞—Ç–∏ –≤—ñ–¥–ø–æ–≤—ñ–¥—ñ –Ω–∞ –æ—Å–Ω–æ–≤—ñ –∑–Ω–∞–π–¥–µ–Ω–∏—Ö –¥–∞–Ω–∏—Ö.",
    "–ü–æ—à—É–∫ —ñ–Ω—Ñ–æ—Ä–º–∞—Ü—ñ—ó –º–æ–∂–ª–∏–≤–∏–π —á–µ—Ä–µ–∑ BM25 —Ç–∞ –≤–µ–∫—Ç–æ—Ä–Ω–∏–π –ø–æ—à—É–∫.",
    "–ì–ª–∏–±–æ–∫—ñ –Ω–µ–π—Ä–æ–º–µ—Ä–µ–∂—ñ –¥–æ–ø–æ–º–∞–≥–∞—é—Ç—å –∞–Ω–∞–ª—ñ–∑—É–≤–∞—Ç–∏ —Ç–µ–∫—Å—Ç.",
    "RAG –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—î –≤–µ–∫—Ç–æ—Ä–Ω—ñ –±–∞–∑–∏ –¥–∞–Ω–∏—Ö –¥–ª—è –ø–æ—à—É–∫—É."
]

# –ü–µ—Ä–µ—Ç–≤–æ—Ä—é—î–º–æ –¥–æ–∫—É–º–µ–Ω—Ç–∏ –≤ –µ–º–±–µ–¥–¥–∏–Ω–≥–∏
doc_embeddings = retrieval_model.encode(documents)
dimension = doc_embeddings.shape[1]

# –°—Ç–≤–æ—Ä—é—î–º–æ FAISS —ñ–Ω–¥–µ–∫—Å
index = faiss.IndexFlatL2(dimension)
index.add(np.array(doc_embeddings))

# –ó–∞–ø–∏—Ç –∫–æ—Ä–∏—Å—Ç—É–≤–∞—á–∞
query = "–Ø–∫ –ø—Ä–∞—Ü—é—î –ø–æ—à—É–∫ –≤ RAG?"
query_embedding = retrieval_model.encode([query])

# –ü–æ—à—É–∫ 3 –Ω–∞–π–±–ª–∏–∂—á–∏—Ö —Ñ—Ä–∞–≥–º–µ–Ω—Ç—ñ–≤
D, I = index.search(np.array(query_embedding), k=3)
retrieved_docs = [documents[idx] for idx in I[0]]

# –ü–µ—Ä–µ—Ä–∞–Ω–∂—É–≤–∞–Ω–Ω—è –∑–Ω–∞–π–¥–µ–Ω–∏—Ö –¥–æ–∫—É–º–µ–Ω—Ç—ñ–≤
scores = reranker.predict([(query, doc) for doc in retrieved_docs])
ranked_docs = [doc for _, doc in sorted(zip(scores, retrieved_docs), reverse=True)]

# –í–∏–≤—ñ–¥ —Ä–µ–∑—É–ª—å—Ç–∞—Ç—ñ–≤
print("–ó–Ω–∞–π–¥–µ–Ω—ñ –¥–æ–∫—É–º–µ–Ω—Ç–∏ (–ø—ñ—Å–ª—è –ø–µ—Ä–µ—Ä–∞–Ω–∂—É–≤–∞–Ω–Ω—è):")
for doc in ranked_docs:
    print(f"- {doc}")
```

**–ü—Ä–∏–∫–ª–∞–¥ –≤–∏–∫–æ–Ω–∞–Ω–Ω—è:**

- –ó–Ω–∞–π–¥–µ–Ω—ñ –¥–æ–∫—É–º–µ–Ω—Ç–∏ (–ø—ñ—Å–ª—è –ø–µ—Ä–µ—Ä–∞–Ω–∂—É–≤–∞–Ω–Ω—è):
  - RAG –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—î –≤–µ–∫—Ç–æ—Ä–Ω—ñ –±–∞–∑–∏ –¥–∞–Ω–∏—Ö –¥–ª—è –ø–æ—à—É–∫—É.
  - –ü–æ—à—É–∫ —ñ–Ω—Ñ–æ—Ä–º–∞—Ü—ñ—ó –º–æ–∂–ª–∏–≤–∏–π —á–µ—Ä–µ–∑ BM25 —Ç–∞ –≤–µ–∫—Ç–æ—Ä–Ω–∏–π –ø–æ—à—É–∫.
  - –ì–ª–∏–±–æ–∫—ñ –Ω–µ–π—Ä–æ–º–µ—Ä–µ–∂—ñ –¥–æ–ø–æ–º–∞–≥–∞—é—Ç—å –∞–Ω–∞–ª—ñ–∑—É–≤–∞—Ç–∏ —Ç–µ–∫—Å—Ç.

#### –†–µ–∞–ª—ñ–∑–∞—Ü—ñ—è —Ä–µ–∑—é–º—É–≤–∞–Ω–Ω—è —Ç–∞ –≥–µ–Ω–µ—Ä–∞—Ü—ñ—ó –≤—ñ–¥–ø–æ–≤—ñ–¥—ñ

–†–µ–∑—é–º—É–≤–∞–Ω–Ω—è –≤–∏–ª—É—á–µ–Ω–∏—Ö –¥–æ–∫—É–º–µ–Ω—Ç—ñ–≤ ‚Äî —Ü–µ –µ—Ç–∞–ø, —è–∫–∏–π –¥–æ–ø–æ–º–∞–≥–∞—î —É—Å—É–Ω—É—Ç–∏ –Ω–∞–¥–ª–∏—à–∫–æ–≤—ñ—Å—Ç—å —Ç–∞ –ø–æ–∫—Ä–∞—â–∏—Ç–∏ —è–∫—ñ—Å—Ç—å –≤—ñ–¥–ø–æ–≤—ñ–¥–µ–π. –†–µ–∫–æ–º–µ–Ω–¥—É—î—Ç—å—Å—è –ø–æ—î–¥–Ω—É–≤–∞—Ç–∏ –µ–∫—Å—Ç—Ä–∞–∫—Ç–∏–≤–Ω—ñ —Ç–∞ –≥–µ–Ω–µ—Ä–∞—Ç–∏–≤–Ω—ñ –ø—ñ–¥—Ö–æ–¥–∏, —Ç–∞–∫–∏–π –ø—ñ–¥—Ö—ñ–¥ –∑–∞–±–µ–∑–ø–µ—á—É—î –≤–∏—Å–æ–∫—É —Ç–æ—á–Ω—ñ—Å—Ç—å —Ç–∞ –µ—Ñ–µ–∫—Ç–∏–≤–Ω—ñ—Å—Ç—å.
